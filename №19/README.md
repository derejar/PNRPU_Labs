# &#8224;Вариант 6:
Реализация стека через классы.
Реализованные функции:
  - top() - доступ к верхнему элементу стека.
  - empty() - проверка стека на наличие в нем элементов. Если стек пуст - возвращает true, иначе - false.
  - size() - размер стека.
  - push() - добавление элемента в стек.
  - pop() - удаление верхнего элемента стека.
  - emplace() - замена верхнего элемента стека.
