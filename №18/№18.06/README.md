# Вариант 6:
#### Класс-контейнер __МНОЖЕСТВО__ с элементами типа int.
Реализовать операции:
  - [] &minus; доступа по индексу.
  - () &minus; определение размера множества.
  - &plus; &minus; объединение множеств.
  - &plus;&plus; &minus; переход к следующему элементу (с помощью класса-итератора).
